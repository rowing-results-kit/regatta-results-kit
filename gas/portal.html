Drive / GitHub 接続設定

秘密値はサーバー側のみで保持します。ここにはマスク値だけが表示されます。

—
📖 取得方法
  1. drive.google.com を開き、大会用フォルダを新規作成します。
  2. 作成したフォルダをクリックして開きます。
  3. ブラウザのアドレスバーに表示された URL をそのままコピーしてこの欄に貼り付けます。ID は自動で取り出されます。
※ このフォルダの中の master/・race_csv/ などのサブフォルダは、初回実行時に自動で作成されます。事前に作る必要はありません。
1つ以上選択してください。
—
—
📖 GITHUB_OWNER / GITHUB_REPO の取得方法
速報サイトのテンプレートから作成した自分のリポジトリを使います。
リポジトリページの URL から読み取ります。 https://github.com/〈オーナー名〉/〈リポジトリ名〉 ↑ これが GITHUB_OWNER ↑ これが GITHUB_REPO 例: https://github.com/sample-rowing/regatta-2027 の場合
　GITHUB_OWNER = sample-rowing　GITHUB_REPO = regatta-2027
—
トークンは保存後マスク表示されず「設定あり」のみ表示します。
📖 取得方法（GitHub fine-grained PAT）
  1. GitHub にログインし、右上のアイコン → Settings を開きます。
  2. 左メニュー最下部の Developer settings をクリックします。
  3. Personal access tokens → Fine-grained tokens を選びます。
  4. Generate new token をクリックします。
  5. Expiration（有効期限）は 90 days を推奨します。
    ※ 90日で自動失効します。失効後はここで再発行・再設定が必要です。
  6. Repository access は「Only select repositories」で対象リポジトリのみを選択します。
  7. Permissions → Repository permissions → Contents を Read and write に設定します。
  8. Generate token をクリックし、表示されたトークン（github_pat_...）を今すぐコピーします。
    ⚠ このページを閉じると二度と表示されません。必ず今コピーしてください。
⚠ 全リポジトリに権限を与える Classic token は使わないでください。

このシステムで何が起きるか

      
管理者が操作するのは2か所だけ。あとは自動で動きます。

      

        

          ①
          
CSV を Drive に入れる

          
計測後、BoatTimer の CSV ファイルを Google Drive の所定フォルダに置くだけ

        

        
▶

        

          ②
          
2分以内に速報サイトへ

          
GAS が2分おきに自動検知し、順位・タイムを公開サイトに反映

        

        
▶

        

          ③
          
PDF はボタン1つ

          
個別レース結果 PDF は自動生成。冊子・一覧は GAS メニューからボタン操作
