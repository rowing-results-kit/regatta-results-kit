導入の流れ

7つのステップで完了します。詳しい手順は次のページで確認できます。

1
3つのアカウントを準備する
Google / GitHub / Cloudflare のアカウントを作ります。全て無料です。
所要時間: 10〜15分
2
キットをコピーする
GitHub からテンプレートを自分のアカウントにコピー（フォーク）します。
所要時間: 5分
3
速報サイトを公開する
Cloudflare に接続するだけで、URLが自動で発行されます。
所要時間: 10分
4
Google ドライブのフォルダを用意する
CSVを入れる専用フォルダを作り、共有設定をします。
所要時間: 5分
5
自動化の仕組みを設定する（GAS）
Google スプレッドシートに付属のスクリプトを貼り付けて、3か所だけ書き換えます。
所要時間: 10〜15分
6
大会の基本情報を登録する
大会名・日程・会場名・種目名などをスプレッドシートに入力します。
所要時間: 10分
7
動作を確認して完了
テスト用のCSVを入れてサイトに反映されるか確認します。
ここまで来たら本番OK
