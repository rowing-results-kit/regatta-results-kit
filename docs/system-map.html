  • scaffold.py サイト・staff・手順書を一括生成

  • generate_race_pdf / _xlsx 競漕記録シート
  • simulate_pipeline / watch GAS無しで再現・リハ

管理者ポータル（AdminPortal）

  • association.json 更新 → ハブ掲載へ反映

① 進行計算は「次フェーズ」

いまのポータルでできるのは進行モデルの選択（registry.json から選び template_id を保存）まで。次ラウンドの自動組分け計算（engine の selectPattern 実行）は次フェーズで、現状は稼働していない。

② 日常で人が触るのは2つだけ

大会当日の運用で人が操作するのは Google Drive（CSV投入・PDF回収） と 管理者ポータル のみ。GAS 3兄弟・サイト反映は自動。ローカルツール（tools/）は技術者がセットアップ・リハで使う裏方。
